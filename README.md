🤖 AI Chatbot Web App

A modern and interactive chatbot built using React + Vite that allows users to send messages and receive instant responses with a smooth UI and real-time typing experience.

🚀 Live Demo

👉 https://chatbot-project-gagan.netlify.app/

✨ Features  
- 🤖 Real-time chatbot interaction with dynamic responses  
- ⚡ Asynchronous API-based response handling  
- 🧠 Clear distinction between user and bot messages  
- 📜 Auto-scrolling chat interface for seamless experience  
- 🎯 Clean and responsive UI  
- 🚀 Fast and optimized performance using Vite  

🛠️ Tech Stack
- ⚛️ React.js
- ⚡ Vite
- 🟨 JavaScript (ES6+)
- 🎨 CSS

 📂 Project Structure  

- src/ → Main source code  
- src/components/ → All React components  
- src/components/ChatInput.jsx → Input box & send logic  
- src/components/ChatMessages.jsx → Chat container  
- src/components/ChatMessage.jsx → Single message UI  
- public/ → Static assets  
- index.html → Root HTML file  
- package.json → Project dependencies & scripts  
- vite.config.js → Vite configuration
